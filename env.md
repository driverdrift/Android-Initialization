[platform-tools](https://developer.android.com/tools/releases/platform-tools)  
[Download SDK Platform-Tools for Windows](https://dl.google.com/android/repository/platform-tools-latest-windows.zip)  
1. Download platform-tools file and extract it.
1. Then add the system environment variables.
1. Run `adb version` in powershell to check if it is configured properly.
1. Click android-os version serveral time to open develop mode.
1. Enable use debugging in develop mode.
1. Run `adb devices` to see if phone connects well.
